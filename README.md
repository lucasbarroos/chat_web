# Chat Web

Simple chat using socket.io and ReactJs. 

## Preview

![Chat Preview](https://i.imgur.com/iXLm5KS.png)

## Installation

Use the package manager npm to install de dependencies:

```bash
npm install
npm run dev
```

Or the yarn:

```
yarn 
yarn dev
```

## Project Technologies
ReactJS, Socket.io.
